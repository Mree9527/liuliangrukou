      # USDT Exchange App Design System
      
      ### **Visual Identity**
      A high-contrast "Neon Fintech" interface optimized for Telegram Mini Apps.
      
      ---
      
      ### **Core UI Components**
      
      | Component | Style Description |
      |-----------|-------------------|
      | **Background** | Deep Charcoal `#181A20` - Professional crypto vibe, low eye strain. |
      | **Cards/Panels** | Glassmorphism Slate `#242730` with subtle borders and glow effects. |
      | **Action Button (Buy)** | Emerald Green Gradient `#00C853`. Large, full-width, high CTR. |
      | **Typography** | Bold numeric font for prices; clean sans-serif for labels. |
      
      ---
      
      ### **App Flow Architecture**
      
      1. **Landing Page**: User sees Live Rate & Balance. "One Thumb" Buy/Sell inputs.
      2. **Payment Modal**: Auto-generated Dynamic QR Code (WallyPay/Bit2Me).
      3. **Success Animation**: Confetti + "USDT Sent!" status.
